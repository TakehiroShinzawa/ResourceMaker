## Version 1.2.0 - 2025-09-24

### 🚀 Release Note
- Added project-level caching support.
- Introduced tooltip resource-assist functionality.
- Added tooltips to improve usability and clarify operations.
- The resource file folder and name can now be customized.
- CSV export is now supported.

- Resource files can now be relocated to any folder and converted between .resx and .resw formats.

- リソースファイルのフォルダと名前をカスタマイズ出来るようになりました。

- リソースファイルを任意の場所に移動したり、リソースの種別(resw/resx)を変換できるようになりました。
